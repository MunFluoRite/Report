Linux의 기초 명령어
-------------
**./a**

a 프로그램 실행

**ls**

파일 리스트 보기

**cd**

디렉토리 변경

**cp**

파일 복사

**mv**

파일 이름 변경 / 파일 위치 변경

**mkdir**

디렉토리 생성 

**rm**

파일 삭제

**pwd**

현재의 디렉토리 경로를 보여주기

**chmod**

파일 권한 변경

**cat** 

파일의 내용을 화면에 출력 / 파일 생성

**whereis**

소스, 실행파일, 메뉴얼 등의 위치를 알려준다

**adduser**

시스템에 사용자 추가
